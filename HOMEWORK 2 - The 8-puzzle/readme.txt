READ ME
1. Open any application that can interpret Python
2. Configure application to run any version greater than Python 3.0
3. Run Main.py

GUI
1. Grid Tab: allows you to manipulate the state of the puzzle
	a. Reset: resets the puzzle to the intial (goal) state
	b. Random State: randomizes the tiles in the puzzle
	c. Input State: customize the position of the tiles manually
2. Puzzle: tiles can be moved around by clicking on tiles adjacent to the blank tile
